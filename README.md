## Featured Work

### AI Messenger Customer Support System

An AI-assisted Facebook Messenger system designed for furniture retail operations.

Key capabilities:

* Customer intent and language detection
* Product catalog and knowledge-base search
* Conversation history and memory
* Message batching and duplicate prevention
* Human-agent takeover and AI pause controls
* Sequential conversation processing
* Supabase-backed runtime state
* Automated reply validation before delivery

**Technologies:** n8n, Supabase, PostgreSQL, Gemini, Meta Messenger API, Docker, Cloudflare Tunnel

---

### Cashier and Sales Reporting Systems

Business tools designed to improve daily transaction recording, reconciliation, and management reporting.

Areas covered:

* Daily cashier sales reports
* Cash, financing, and payment-method separation
* POS data validation
* Sales and collection reconciliation
* Automated Google Sheets processing
* Branch and transaction-level reporting

**Technologies:** Google Sheets, Apps Script, SQL, C#, PHP, JavaScript
